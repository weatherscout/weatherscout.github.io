WeatherScout Docs

A comprehensive guide to the WeatherScout interface and features.

Core Features

WeatherScout is a high-performance weather visualization tool built on MapLibre GL. It prioritizes speed, clarity, and real-time data accuracy.

  • Dynamic Interface: Glassmorphism UI that stays out of the way of the data.
  • Smart Navigation: Right-click anywhere to instantly lock onto the nearest radar site.
  • Auto-Update: All data feeds (Radar, Alerts, SPC) refresh automatically in the background.

Radar Capabilities

National Mosaic

The default view provides a smoothed reflectivity composite of the entire United States. Managed via the Mosaic toggle in the sidebar.

Single-Site Precision

Access high-resolution data by clicking any radar station dot. The system supports WSR-88D (Standard) and TDWR (Airport-based) stations.

  • Reflectivity: Detects precipitation intensity.
  • Velocity: Detects wind movement (crucial for rotation tracking).
  • Radar Info Pill: Displays the current Station ID and Product at the top left. Click it to cycle products.

Severe Alerts

WeatherScout processes live NWS alerts with specific visual logic:

  • High-Threat Highlighting: Warnings with "Catastrophic" or "Destructive" tags receive thicker borders and priority map flashing.
  • Interaction: Click a map alert to view the "Full Text" bulletin or a "Pager" if multiple alerts overlap.
  • Zone Alerts: Toggles broad alerts (Watches/Advisories) to keep the map clean during active outbreaks.

SPC Outlooks

Visualize Storm Prediction Center forecasts directly on the map. Select a Day (1-8) and a Risk Type (Categorical, Tornado, Hail, Wind, or Probabilistic).

Analysis: Click any outlook area to read the full technical discussion from SPC meteorologists.

URL Parameters

Control the app state via the URL string using ?s= for settings and ?l= for location.

FlagDescription
ftr / xfrFly to Radar (On / Off)
alt / xalAlerts Visibility (On / Off)
zna / xzaZone Alerts (On / Off)
dbg / xdbDebug Mode (On / Off)
sav / xsvSave Settings to Cache (On / Off)
r[ID]Auto-load Reflectivity for ID (e.g. rKTLX)
l=[City]Geocode and fly to location (e.g. l=Norman-OK)

App Settings

  • Site Selection: Filter map dots by station type (WSR-88D vs TDWR).
  • Fly to Radar: Automatically zooms into a station when you select it.
  • Debug Mode: For developers. Unlocks the Shift + U upload feature for testing GeoJSON payloads.
  • Save Settings: Stores your zoom, position, and active layers in browser storage.
